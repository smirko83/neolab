имеется app.jar файл.
Для его запуска используется команда

java -jar app.jar some_out_file "Service is working!" 

Необходимо сделать init файл для старт/стоп программы как сервиса в linux.
init скрипт сделать в стиле SysV или systemd.
При старте возвращать успех только после того, как пройдет 5 сек и сервис не упадет. 
Если сервис упадет раньше, то сразу возращать ошибку без ожидания окончания 5 сек.
Предусмотреть возможность задания другой java через конфиг и задавать путь к выходному файлу также, через конфиг.
